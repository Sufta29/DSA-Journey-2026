# Student Management System (C++)

A menu-driven Student Management System built using C++ and STL.

## Features
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student

## Concepts Used
- C++
- Structures
- Vectors (STL)
- Functions
- Switch Case

## How to Run

```bash
g++ main.cpp -o sms
./sms
```
